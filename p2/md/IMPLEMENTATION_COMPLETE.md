## 🚀 How to Use

### Quick Start (Automated)
```bash
# Windows
cd p2
setup_frontend.bat

# Mac/Linux
cd p2
chmod +x setup_frontend.sh
./setup_frontend.sh
```

### Manual Setup
```bash
cd p2/frontend
npm install
npm run dev
```

### Access Application
```
http://localhost:5173
```
