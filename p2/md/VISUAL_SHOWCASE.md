# 🎨 Visual Showcase - InterviewAI Frontend

## Design Overview

A modern, minimal, premium-looking interface inspired by Stripe, Linear, and Notion. Dark mode first with glassmorphism effects, smooth animations, and a professional aesthetic.


```
┌──────────────────────┐
│ ████████████████████ │  Blue → Purple
│ ▶ Start Interview   │  Glow Effect
│ ████████████████████ │  Hover: Scale 1.05
└──────────────────────┘
```

### Animations
```
Fade In:     opacity: 0 → 1, y: 20 → 0
Hover Lift:  y: 0 → -5
Scale:       scale: 1 → 1.05
Duration:    300-600ms
```

## ✨ Interactive Elements

### Hover States
```
Card:    Lift up 5px + border glow
Button:  Scale 1.05 + glow increase
Link:    Color change (gray → white)
```

### Click States
```
Button:  Scale 0.95 (tap feedback)
Card:    Subtle press effect
```

### Loading States
```
Upload:  Progress bar animation
Data:    Skeleton screens
Button:  Spinner icon
```
