## ⚡ Quick Setup

### Windows
```bash
cd p2
setup_frontend.bat
```

### Mac/Linux
```bash
cd p2
chmod +x setup_frontend.sh
./setup_frontend.sh
```

### Manual
```bash
cd p2/frontend
npm install
npm run dev
```

Then open: **http://localhost:5173**
