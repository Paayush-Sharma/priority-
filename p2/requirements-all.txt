# ============================================
# TROUBLESHOOTING
# ============================================
# 
# Issue: "FFmpeg not found"
# Solution: Install FFmpeg and add to PATH, restart terminal
# 
# Issue: "Module not found"
# Solution: pip install -r requirements-all.txt --force-reinstall
# 
# Issue: "Port already in use"
# Solution: Change port or kill process
#   Windows: netstat -ano | findstr :8000 && taskkill /PID <PID> /F
#   Mac/Linux: lsof -ti:8000 | xargs kill -9
# 
# Issue: "numpy version conflict"
# Solution: pip install "numpy>=1.24.0,<2.0.0" --force-reinstall
# 
# Issue: "Virtual environment won't activate" (Windows PowerShell)
# Solution: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
# 
# For more help, see TROUBLESHOOTING_INSTALL.md
